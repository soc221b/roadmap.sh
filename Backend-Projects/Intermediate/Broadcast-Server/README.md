# Broadcast Server

https://roadmap.sh/projects/broadcast-server

## Getting Started

```sh
$ git clone https://github.com/soc221b/roadmap.sh.git
$ cd roadmap.sh/Backend-Projects/Intermediate/Broadcast-Server
$ npm ci
```

## Run Locally

```sh
$ node main.ts start
$ node main.ts connect --user Your-name
```
